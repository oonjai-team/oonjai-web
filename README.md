# Oonjai Web
A frontend repository for Oonjai

## Prerequisite

This project is intended to be run with **[Bun](https://bun.sh/)** (a fast JavaScript runtime + package manager). Using Bun helps ensure consistent installs and matches the project’s lockfile.

### Install Bun

Choose one option:

- **macOS / Linux (recommended):**
  ```bash
  curl -fsSL https://bun.sh/install | bash
  ```

- **Windows:**
  Follow the official installer instructions: https://bun.sh/


Happy coding!
